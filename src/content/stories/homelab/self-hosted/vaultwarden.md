*Self-Hosted Password Manager*

Vaultwarden is an unofficial Bitwarden-compatible server written in Rust, perfect for self-hosted password management with minimal resource requirements.

## Key Features

- Password vault storage
- Browser extensions
- Mobile apps for iOS and Android
- Two-factor authentication
- Secure sharing
- Self-host with low resources

## Why I Use It

Storing passwords in a service someone else controls has always bothered me, and Vaultwarden is the answer to that. It is a lightweight Bitwarden-compatible server, which means I get to use all the official Bitwarden browser extensions and mobile apps while the actual vault lives on my own hardware. The Bitwarden ecosystem is mature and the apps are excellent — I am not giving anything up by running my own backend. Passwords, passkeys, secure notes, and shared credentials all stay within my network.
