*Game Boy Advance Emulator*

nextGBA is a web-based Game Boy Advance emulator that allows you to play classic GBA games directly in your browser.

## Key Features

- Browser-based emulation
- Save state support
- ROM loading
- Touch and keyboard controls
- No installation required
- Cross-platform compatibility

## Why I Use It

Not everything in the homelab has to be serious. nextGBA is just a fun one — a browser-based GBA emulator that I can load up from any device without installing anything. The Game Boy Advance had an incredible library, and being able to revisit those games through a browser with save states is a nice bit of nostalgia. It runs well, requires zero setup once it is hosted, and is a good reminder that self-hosting does not always have to be about infrastructure.
