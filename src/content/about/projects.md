A few of the projects I'm proud of, with full write-ups elsewhere on the site:

- **[NVD Scanner](/stories/cyber/projects/nvd-scanner/)** — a JavaScript tool that queries the NVD API by keyword and returns CVE details, severity, proof-of-concept references, and CWE links.
- **[Machine Learning Network Traffic Analysis](/stories/cyber/projects/jetson-orin-netflow-ml/)** — ML-powered NetFlow collection and anomaly detection running on an NVIDIA Jetson Orin Nano Super.
- **[Proxmox Cyber Lab](/stories/cyber/cyber-labs/proxmox-cyber-lab/)** — an isolated Active Directory lab in Proxmox with Windows domain controllers, Tailscale VPN access, and PowerShell-driven management.
- **[Automated Site Deployments with Jenkins](/stories/homelab/tools/jenkins/)** — a CI/CD pipeline that builds and deploys this site automatically from Forgejo webhooks.
- **[The Homelab](/homelab/)** — the multi-node Proxmox environment behind most of these write-ups, from the [hardware](/homelab/hardware/) to the [self-hosted apps](/homelab/self-hosted/) running on it.
